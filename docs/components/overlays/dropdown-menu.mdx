# DropdownMenu

`DropdownMenu` is an overlay component for displaying a list of actions or options in a dropdown menu.

## Import

```tsx
import { DropdownMenu } from '@radix-ui/themes-native';
```

## Usage

```tsx
import { DropdownMenu, IconButton, Text } from '@radix-ui/themes-native';

<DropdownMenu.Root>
  <DropdownMenu.Trigger>
    <IconButton>
      <Icon name="more-vertical" />
    </IconButton>
  </DropdownMenu.Trigger>

  <DropdownMenu.Content>
    <DropdownMenu.Item>
      <Text>Edit</Text>
    </DropdownMenu.Item>
    <DropdownMenu.Item>
      <Text>Duplicate</Text>
    </DropdownMenu.Item>
    <DropdownMenu.Separator />
    <DropdownMenu.Item color="red">
      <Text>Delete</Text>
    </DropdownMenu.Item>
  </DropdownMenu.Content>
</DropdownMenu.Root>
```

## Components

### DropdownMenu.Root

The root container component that manages menu state.

### DropdownMenu.Trigger

Element that opens the menu.

### DropdownMenu.Content

The menu content container.

### DropdownMenu.Item

Individual menu item.

### DropdownMenu.Separator

Visual separator between items.

### DropdownMenu.Label

Label for a group of items.

### DropdownMenu.Group

Group of related items.

### DropdownMenu.CheckboxItem

Checkable menu item.

### DropdownMenu.RadioGroup

Group of mutually exclusive items.

### DropdownMenu.RadioItem

Radio-style menu item.

### DropdownMenu.Sub

Nested submenu.

### DropdownMenu.SubTrigger

Trigger for a submenu.

### DropdownMenu.SubContent

Content for a submenu.

## Props

### DropdownMenu.Root

<PropTable>
  <Prop name="children" type="ReactNode">
    Menu components.
  </Prop>
  <Prop name="open" type="boolean">
    Controlled open state.
  </Prop>
  <Prop name="onOpenChange" type="(open: boolean) => void">
    Callback when open state changes.
  </Prop>
</PropTable>

### DropdownMenu.Content

<PropTable>
  <Prop name="children" type="ReactNode">
    Menu items.
  </Prop>
  <Prop name="style" type="StyleProp<ViewStyle>">
    Custom styles.
  </Prop>
  <Prop name="side" type="'top' | 'right' | 'bottom' | 'left'" default="'bottom'">
    Preferred side to render.
  </Prop>
  <Prop name="sideOffset" type="number">
    Distance from trigger.
  </Prop>
  <Prop name="align" type="'start' | 'center' | 'end'" default="'start'">
    Alignment relative to trigger.
  </Prop>
</PropTable>

### DropdownMenu.Item

<PropTable>
  <Prop name="children" type="ReactNode">
    Item content.
  </Prop>
  <Prop name="onSelect" type="() => void">
    Callback when item is selected.
  </Prop>
  <Prop name="disabled" type="boolean" default="false">
    Whether the item is disabled.
  </Prop>
  <Prop name="color" type="string">
    Item color (e.g., "red" for destructive actions).
  </Prop>
  <Prop name="style" type="StyleProp<ViewStyle>">
    Custom styles.
  </Prop>
</PropTable>

## Examples

### Basic Menu

```tsx
<DropdownMenu.Root>
  <DropdownMenu.Trigger>
    <Button>Options</Button>
  </DropdownMenu.Trigger>

  <DropdownMenu.Content>
    <DropdownMenu.Item>Edit</DropdownMenu.Item>
    <DropdownMenu.Item>Duplicate</DropdownMenu.Item>
    <DropdownMenu.Item>Share</DropdownMenu.Item>
  </DropdownMenu.Content>
</DropdownMenu.Root>
```
