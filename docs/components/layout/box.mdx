# Box

`Box` is a primitive container component that provides access to all styling props. It's the building block for creating custom components and layouts.

## Import

```tsx
import { Box } from '@radix-ui/themes-native';
```

## Usage

```tsx
import { Box, Text } from '@radix-ui/themes-native';

<Box padding={4} backgroundColor="gray.2">
  <Text>Content inside a box</Text>
</Box>
```

## Examples

### Basic Container

```tsx
<Box padding={4} backgroundColor="gray.2">
  <Text>Simple container</Text>
</Box>
```

### Card-like Box

```tsx
<Box
  padding={4}
  backgroundColor="gray.2"
  borderRadius={3}
>
  <Text>Card content</Text>
</Box>
```

### Flex Container

```tsx
<Box display="flex" flexDirection="row" gap={3}>
  <Text>Item 1</Text>
  <Text>Item 2</Text>
  <Text>Item 3</Text>
</Box>
```

### With Border

```tsx
<Box
  padding={4}
  borderWidth={1}
  borderColor="gray.6"
  borderRadius={2}
>
  <Text>Bordered box</Text>
</Box>
```

### Responsive Dimensions

```tsx
<Box width="100%" height={200} backgroundColor="blue.3">
  <Text>Full width, fixed height</Text>
</Box>
```

### With Shadow

```tsx
<Box
  padding={4}
  backgroundColor="gray.1"
  borderRadius={3}
  elevation={2}
>
  <Text>Box with shadow</Text>
</Box>
```

## Shorthand Props

Box supports shorthand props for common styles:

```tsx
// Padding
<Box p={4} />          // padding
<Box px={4} />         // paddingHorizontal
<Box py={4} />         // paddingVertical
<Box pt={4} />         // paddingTop
<Box pb={4} />         // paddingBottom
<Box pl={4} />         // paddingLeft
<Box pr={4} />         // paddingRight

// Margin
<Box m={4} />          // margin
<Box mx={4} />         // marginHorizontal
<Box my={4} />         // marginVertical
<Box mt={4} />         // marginTop
<Box mb={4} />         // marginBottom
<Box ml={4} />         // marginLeft
<Box mr={4} />         // marginRight
```
